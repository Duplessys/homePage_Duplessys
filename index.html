Quem somos

A Duplessys é uma empresa especializada em soluções de dados e análise de informações, dedicada a ajudar empresas a organizar, consolidar e transformar dados em informações estratégicas para apoiar decisões de negócio.

Nascemos da vontade de tratar o mercado de dados e análise de informações de forma individualizada. Sabemos que cada empresa possui suas próprias dores e seu próprio contexto, e que soluções padronizadas dificilmente atendem todos os cenários.

A nossa abordagem consiste em primeiro entender o problema para depois desenvolver soluções ou propor desenvolver soluções e propor as tecnologias mais adequadas às necessidades específicas de cada cliente.

O que fazemos

  • Soluções em banco de dados
    • Modelagem e tratamento de dados
    • Construção de bases de dados em:
      • SQL
      • Oracle
      • BigQuery
  • Desenvolvimento de indicadores e dashboards
    • Modelagem de indicadores
    • Desenvolvimento de dashboards
    • Ferramentas:
      • Power BI
      • Excel
      • Google Data Studio
